## 题解1

```java
class Solution {
    public int evalRPN(String[] tokens) {
        Deque<Integer> stack = new LinkedList<>();
        int a, b;

        for (String cur : tokens) {
            if (cur.equals("+")) {
                b = stack.pop();
                a = stack.pop();
                stack.push(a+b);
            } else if (cur.equals("-")) {
                b = stack.pop();
                a = stack.pop();
                stack.push(a-b);
            } else if (cur.equals("*")) {
                b = stack.pop();
                a = stack.pop();
                stack.push(a*b);
            } else if (cur.equals("/")) {
                b = stack.pop();
                a = stack.pop();
                stack.push((int)a/b);
            } else {
                stack.push(Integer.valueOf(cur));
            }
        }

        return stack.pop();
    }
}
```
