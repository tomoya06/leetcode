## 题解1

做过的题目，差点忘了怎么做。

```java
class Solution {
    public int[][] merge(int[][] intervals) {
        Arrays.sort(intervals, (int[] a, int[] b) -> {
            if (a[0] != b[0]) {
                return a[0] - b[0];
            }
            return a[1] - b[1];
        });

        Deque<int[]> res = new LinkedList<>();

        for (var inter : intervals) {
            if (res.isEmpty()) {
                res.add(inter);
                continue;
            }
            if (inter[0] <= res.getLast()[1]) {
                res.getLast()[1] = Math.max(res.getLast()[1], inter[1]);
            } else {
                res.add(inter);
            }
        }

        return res.toArray(new int[res.size()][2]);
    }
}
```
