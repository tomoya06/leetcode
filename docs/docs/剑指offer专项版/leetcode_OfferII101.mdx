## 题解1

你才简单，你全家都简单

```java
class Solution {
    public boolean canPartition(int[] nums) {
        int sum = 0;
        for (int num : nums) {
            sum += num;
        }
        if (sum % 2 != 0) {
            return false;
        }

        int half = sum/2, n = nums.length;
        int[] dp = new int[half+1];
        dp[0] = 1;
        for (int i=0; i<n; i++) {
            int num = nums[i];
            for (int s=half; s>=num; s--) {
                dp[s] += dp[s-num];

                if (s==half && dp[s] > 0) {
                    return true;
                }
            }
        }

        return false;
    }
}

```
